Medium Write-up Bug Hunting

  • Category: Project
  • Topic: Bug Hunting & Web Security
  • Platform: Medium
  • Project URL: Read on Medium

Detail

Saya menulis artikel pertama saya di Medium dengan judul Exploiting an Exposed .git: Gaining Access to Source Code and Database. Tulisan ini membahas pengalaman saya saat melakukan bug hunting pada sebuah aplikasi web dan menemukan direktori .git yang dapat diakses secara publik.

Dalam proses eksplorasi target, saya mencoba mengakses beberapa path umum yang berpotensi menyimpan informasi sensitif. Salah satunya adalah .git/config, yang seharusnya tidak dapat diakses oleh publik. Dari temuan tersebut, saya dapat mengidentifikasi adanya risiko kebocoran konfigurasi Git pada subdomain target.

Setelah mengetahui bahwa folder .git terbuka, saya melakukan proses rekonstruksi repository menggunakan GitTools untuk menganalisis source code yang berhasil diekstrak. Dari hasil analisis, ditemukan file database yang berisi kredensial seperti akun admin, staff, dan data penting lainnya. Informasi tersebut kemudian berhasil digunakan untuk login ke sistem target.

Write-up ini menjadi dokumentasi pembelajaran mengenai dampak serius dari kesalahan konfigurasi server, khususnya ketika folder .git terekspos di lingkungan produksi. Melalui tulisan ini, saya juga membagikan pentingnya validasi konfigurasi deployment dan pembatasan akses terhadap file sensitif pada aplikasi web.
