Broken Authentication Write-up

  • Category: Project
  • Topic: Bug Hunting & Web Security
  • Platform: Medium
  • Project URL: Read on Medium

Detail

Saya menulis artikel Medium dengan judul Broken Authentication: Bypass OTP with JSON Manipulation . Tulisan ini membahas pengalaman saya saat melakukan bug hunting dan menemukan celah keamanan pada sistem verifikasi OTP.

Pada temuan ini, terdapat kelemahan pada logic API yang memungkinkan proses verifikasi OTP dapat dilewati melalui manipulasi response JSON. Celah tersebut menyebabkan akun dapat dibuat tanpa memasukkan OTP yang valid, sehingga berpotensi menimbulkan risiko pembuatan akun tidak sah.

Write-up ini menjelaskan bagaimana kesalahan kecil pada proses validasi authentication dapat berdampak serius terhadap keamanan aplikasi. Melalui dokumentasi ini, saya menyoroti pentingnya validasi di sisi server, terutama pada fitur sensitif seperti OTP verification, account registration, dan authentication flow.

Dari pengalaman ini, saya mendapatkan pemahaman lebih dalam mengenai broken authentication, keamanan API, serta pentingnya memastikan bahwa keputusan keamanan tidak hanya bergantung pada response yang dapat dimanipulasi di sisi client.
